##2. 如何实现多设备同步hexo搭建Github博客##
在Github仓库的master分支上是我们上传的博客文件，这个博客文件是不包含核心哦配置的。要实现多设备同步hexo搭建Github博客，我们需要新建分支，此处我们新建分支hexo，然后使用git命令将核心哦配置的Github工程文件上传到新建的分支上

在本地博客根目录下（本文中即指hexo文件夹）使用git指令上传项目到Github。
